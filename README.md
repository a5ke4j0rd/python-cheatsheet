# Большая шпаргалка по языку Python

Репозиторий где описаны все темы языка программирования Python, вплоть до тонкостей языка.<br>
Здесь расположены статьи из различных источников, таких как: [Хабр](https://habr.com), [Яндекс Практикум](https://practicum.yandex.ru), [ProgLib](https://proglib.io/), [Tproger](https://tproger.ru/) и другие.<br> 
Ссылки на авторов и источники будут прикрепляться в конце каждого поста.<br>

### 2. **Управляющие конструкции**

- **Условные операторы**
  - `if`, `elif`, `else`
  - Вложенные условия
  - Тернарный оператор: `x = a if condition else b`

  - Использование `bool()` для неявного приведения
- **Циклы**
  - `for` (итерирование по спискам, строкам, `range()`)
  - `while`
  - Управление циклами: `break`, `continue`, `pass`, `else`
  - Эффективное итерирование (`enumerate`, `zip`, `itertools`)
  - Циклы в одну строку (list comprehensions)
  - Итерационные протоколы (`__iter__`, `__next__`)
- **Исключения в управлении потоком**
  - Паттерн "Look Before You Leap" vs EAFP
  - Использование исключений для контроля потока

### 4. **Функции**

- **Основы**
  - Определение функций (`def`, `lambda`)

  - Функции высшего порядка (`map`, `filter`, `reduce`)

  - Рекурсия и мемоизация (`functools.lru_cache`)
- **Функциональное программирование**

- **Динамическое создание**
  - `exec()` и `eval()`

- **Интроспекция**
  - Инспекция аргументов (`inspect` модуль)
  - Декораторы для анализа производительности
